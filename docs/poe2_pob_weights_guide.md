# ⚔️ PoE2 Huntress向け PoB `Adjust search weights` 最適化ガイド

このドキュメントは、PoB（Path of Building）のトレード検索機能における **`Adjust search weights`（検索重みの調整）** の全設定項目を調査し、現在の「Evasion（回避）ベースへの移行」および「T4マップでの生存力確保」に特化した最適な重み付け設定をまとめたものです。

---

## 1. `Adjust search weights` とは？

PoBのトレード検索において、取得した装備候補が**「どれだけ自分のキャラクターを強化するか」をスコア化するための重み付け設定**です。
ここで設定した各項目の数値（Weight）に基づき、アイテムを装備した際のステータス変化量（Δ値）から総合スコア（Weighted Sum）が計算され、最も効率の良い装備が上位にソートされます。

$$\text{総合スコア} = \sum (\Delta\text{ステータス} \times \text{設定した重み})$$

---

## 2. 主要設定項目の調査とビルドへの影響

| 項目名 (PoB表記) | 役割と説明 | Huntress（Twister）における重要度と影響 |
| :--- | :--- | :--- |
| **`Full DPS`** | 与ダメージの総合上昇量。 | **中**：戦闘を早く終わらせるために必要ですが、現在は死なないための「防御」を優先すべきなため、重みは控えめに設定します。 |
| **`Effective Hit Pool` (EHP)** | 敵の攻撃にどれだけ耐えられるかの総合耐久値。 | **極高（最優先）**：T4での即死を防ぐための最重要指標。ライフ、耐性、回避、デフレクションなど全てを加味した耐久計算の基本となります。 |
| **`Evasion Rating`** | 回避値の上昇量。 | **極高**：現在の「Evasionベース防具への移行」を加速させるため、個別に高い重みを設定します。これにより、Evasionが高い装備が優先して選ばれます。 |
| **`maximum Life`** | 最大ライフの純増量。 | **高**：Evasionと並んで一撃死を防ぐための基礎体力です。常に高い優先度を維持します。 |
| **`Elemental Resistances`**<br>(Fire / Cold / Lightning) | 各属性耐性の合計値。 | **高（キャップ未満の時のみ）**：耐性上限（75%）に達するまでは極めて重要です。上限に達した後は自動的に評価が下がります。 |
| **`Chaos Resistance`** | カオス耐性の上昇量。 | **中**：現在は0%と非常に低いため、少しだけプラス評価を与えることで、カオス耐性付きの装備が評価されやすくなります。 |
| **`Dexterity`** | 器用さ（Dex）の純増量。 | **中（要求値 157 未満の時のみ）**：強力な物理槍（要求Dex 157）を装備するために現在27不足しています。不足分を補うために一時的に重みを付けます。 |
| **`Intelligence`** | 知性（Int）の純増量。 | **中（オーラ要求値未満の時のみ）**：高レベルオーラを使用するためのInt不足（現在50、目標130）を補うために、わずかに重みを付けます。 |

## 3. 【現在推奨】生存・Evasionベース移行期の最適重み付け設定

現在の「T4で死ぬのを防ぎつつ、Evasion装備を集める」フェーズに最も合致した重み付けの数値設定です。この数値をPoBの `Adjust search weights` 画面で入力してください。

### 📋 推奨設定値テーブル

| 項目 (Stat) | 設定する重み (Weight) | 設定の意図・理由 |
| :--- | :---: | :--- |
| **`Effective Hit Pool`** | **`1.0`** | **耐久力を最優先**にするため、最大値を設定します。 |
| **`Evasion Rating`** | **`0.8`** | **Evasion（回避）ベース防具を強力に推進**するため、個別に高数値を割り当てます。 |
| **`maximum Life`** | **`0.8`** | ライフによる一撃死防止効果を高く評価します。 |
| **`Cold Resistance`** | **`0.5`** | 現在65%で不足している冷気耐性を補うため。 |
| **`Lightning Resistance`** | **`0.5`** | 現在56%で不足している雷耐性を補うため。 |
| **`Chaos Resistance`** | **`0.3`** | カオス耐性0%を少しでも改善させるため。 |
| **`Full DPS`** | **`0.3`** | 火力も最低限確保しますが、防御よりは優先度を下げます。 |
| **`Dexterity`** | **`0.3`** | 物理槍の要求値（157）に到達するためのDex補正。 |
| **`Intelligence`** | **`0.2`** | オーラ運用のためのInt補正。 |

## 4. 【将来用】エンドゲーム・火力クリティカル移行期の重み付け設定

耐性が完全にキャップされ、防具がすべてEvasionベースになり、クリティカルビルドへ移行する段階（Lv80以降）での設定です。

### 📋 将来の推奨設定値テーブル

| 項目 (Stat) | 設定する重み (Weight) | 設定の意図・理由 |
| :--- | :---: | :--- |
| **`Full DPS`** | **`1.0`** | ダメージ効率を最大化し、マップ周回速度を上げるため最優先。 |
| **`Crit Chance / Crit Multiplier`** | **`0.8`** | クリティカルビルドの威力を伸ばす装備を優先。 |
| **`Effective Hit Pool`** | **`0.5`** | 最低限の耐久力は維持しますが、火力より優先度を下げます。 |
| **`Evasion Rating`** | **`0.5`** | 既にEvasionが十分なため、増加効率を少し落とします。 |
| **`maximum Life`** | **`0.5`** | 基礎ライフの補強。 |
| **`Chaos Resistance`** | **`0.3`** | 赤マップ以降のカオスダメージ対策。 |

## 5. PoBでの入力手順

1. 画像右上の **`Adjust search weights`** をクリックします。
2. 表示されたウィンドウの左下などにある **`Add Stat`** ボタンを押します。
3. 検索窓に上記の「項目名」を入力して選択し、リストに追加します。
4. 追加した各項目の **`Weight`** カラムに、推奨設定値テーブルの数値を入力します。
5. **`Done`** または **`Save`** を押して画面を閉じます。
6. この状態で、購入したい部位の `Find best` を実行します。

## 1. ファームの概要とパッチによる変更点

### 📢 アップデートによる変更点
*   **バグ修正**: 以前は Delirium のエリート（レア/マジック）モンスターがスポンせず、ノーマル（白）モブばかりが出現するバグがありました。最新パッチでこれが修正され、**大量のエリートモンスターがスポンする**ようになりました。
*   **200% Delirious**: 現在の仕様では、複数の Delirium タブレットを組み合わせることで最大 200% の Deliriousness（霧の濃度）に到達できます（通常は4枚使用）。これにより画面が分裂ミラー（Fracturing Mirrors）で埋め尽くされます。

### 💰 収益性と期待リターン
*   **投資**: 1マップあたり約 **1.5 Divine** 程度の低投資。
*   **期待収益**: 1マップあたり平均 **5〜8 Divine** の総価値（上振れ時には **10〜15 Divine** も容易に達成可能）。
*   **Simulacrum**: 1ランで Simulacrum が 1〜2 個確定で完成します（動画公開時点で1個約 2.8〜3 Divine の価値）。
*   **高額ドロップ狙い**: Simulacrum 自身のランナー用ドロップも美味しく、**Voices**（最高で 1,200 Divine / 7ミラー相当）や **Megalomaniac**（200 Divine以上で売れるものも）などの超高額ユニークも狙えます。また、Delirium ユニークボス（Kosis, Omniphobia）からのドロップも非常に優秀です。

## 2. タブレットMod (Tablet Modifiers) の選定

タブレットの Mod 構成がこのファームの成否を分けます。ファームの目的に応じて「通常ファーム用（分裂ミラー優先）」と「ボス狙いファーム用（鏡の破片＋ボス出現率優先）」で使い分けます。

| 優先度 | 明示モディファイア (Explicit Mod) | 解説・効果 |
| :---: | :--- | :--- |
| **最優先<br>(通常ファーム)** | `Delirium Fog in Map spawns (15–30)% increased Fracturing Mirrors`<br>(マップ内の霧で分裂ミラー出現率増加) | **雑魚・エリートを増やしてベースドロップを増やす最重要Mod**。通常モブの密度を極限まで高め、カレンシーやシミュラクラムの破片のドロップ数を底上げします。これ単体の石板は比較的安価です。 |
| **最優先<br>(ボスファーム)** | `Delirium Fog in Map spawns (15–30)% increased Mirror Shards`<br>(マップ内の霧で鏡の破片出現率増加) | **ボスファームには必須のMod**。ボス（Kosis/Omniphobia）のスポン契機となる「Deceptive/Capricious Shards」や、霧を一時停止させる「Escalation Shards」などの特殊シャードの出現数を増やします。 |
| **ボス用必須** | `Delirium Encounters in Map are (X)% more likely to spawn Unique Bosses`<br>(マップ内のエンカウントでユニークボスが出現しやすくなる) | 鏡の破片オブジェクトを壊した際に、ユニークボスが出現する確率を高めます。**「鏡の破片増加」Modとセットで運用することで初めて高い効果を発揮します**（シャード数が少ないとボストリガーの試行回数自体が少なくなってしまうため）。 |
| **推奨** | `(15–30)% increased Stack size of Simulacrum Splinters found in Map`<br>(シミュラクラムの破片のスタックサイズ増加) | 1マップで 500〜800 個の Splinter をドロップさせ、Simulacrumの完成を早めるために非常に有用です。 |
| **推奨** | `Delirium Fog in Map dissipates (20–30)% slower`<br>(霧の消退速度低下) / `lasts (6–12) additional seconds`<br>(霧の持続時間追加) | 霧を維持しやすくなり、マップを隅々までファームする際の安定性が大幅に向上します。 |
| **その他** | `Effectiveness` (効果) / `increased number of Rare Monsters` (レアモンスター数) / `increased Rarity` (アイテムレアリティ) 等 | モンスターの強さ、出現数、ドロップ品質をさらにスケールさせるために有効な優良Prefixです。 |

#### 💡 もっともうまく探すための「3つの必須フィルタ設定（手動調整）」

手動で検索条件を細かく調整したい場合のポイントです。


1.  **残り使用回数 (Remaining Uses) を必ず `10` に制限する【最重要】**
    *   Precursor Tablet は使用制限（最大10回）があります。トレードサイトでは残り使用回数が `1` や `2` に減った使い古しが安値で出品され、検索上位を埋め尽くします。
    *   **対策**: `Stat Filters` に `remaining` または `uses` と入力し、**`# uses remaining` フィルタを追加**して、**Min（最小値）に `10`（または `9`）を入力**して検索してください。これを行うだけで、1回あたりのコストパフォーマンスが最も良い「新品の石板」だけをフィルタリングできます。
2.  **必須Modを固定する (And グループ)**
    *   通常ファームなら `Stat Filters` に `fracturing` と入力し、`Delirium Fog in Map spawns #% increased Fracturing Mirrors` を選択します。
    *   ボスファームなら `Stat Filters` に `mirrorshards` と入力し、`Delirium Fog in Map spawns #% increased MirrorShards` を選択します。
3.  **優良Modの組み合わせを探す (Count グループの活用)**
    *   複数の良Mod（`simulacrum splinters stack size`, `unique bosses`, `effectiveness` など）のいずれかが付いている複合石板を探す場合、Stat Filters のタイプを **`Count`** に変更し、**`Minimum` を `1` または `2`** に設定した上で、それらのModをグループ内に追加します。
    *   これにより、「分裂ミラー増加（または鏡の破片増加）」を必須としつつ、「Splinterスタックサイズ増加」や「ユニークボス出現率アップ」のいずれかが付いた優秀な石板を網羅的に探すことができます。

## 4. マップ・アトラスと Jado (ジャド) のセットアップ

### 🗺️ マップの設定
*   **City Biome (市街地バイオーム) が必須**: アトラス上で市街地ノードの恩恵が非常に大きいため、基本は市街地ゾーンで実行します。
*   **Delirium アトラス**: 最大化してください。

### 🦅 Jado のセットアップ
1.  **Unexpected Missions**: 市街地マップのレイアウトを別のものに交換するために採用（市街地は通常レイアウトが走りづらいため）。また、これによって **Versium Remnants** の出現チャンスも生まれます。
2.  **Partial Translations**: **最重要**。タブレットの明示モディファイア（explicit modifiers）の効果をバフし、分裂ミラーの数を極限まで増やします。
3.  **Ancient Activations**: 分裂ミラーの起動を快適にするために使用。
4.  **4つ目のスロット**: 自由枠（高Modマップを走るなら **Doryani** なども選択肢）。

## 5. ファームの進め方 (Fog のセットアップ)

1.  Delirium タブレットをセットしてマップに入り、出現したミラーをクリックして起動します。
2.  ゾーン（City Zone）を選択して Delirious（霧の濃度）を上げます。
3.  **注意点**: 最初（1〜2マップ目）は Deliriousness が 0% から始まるため、霧のジュースアップ（濃度上げ）が必要です。最初の数マップはドロップがあまり美味しくありませんが、1〜2マップ走れば急速に濃度が最大化します。

## 6. 実践での注意点と難易度対策

*   **強力なビルドが必須**: モンスターの密度と火力が非常に高いため、弱いビルドで突入すると瞬殺されます。
    *   **対策**: ビルドのパワーが足りない場合は、マップの Mod 数を抑える、タブレットを 2〜3 枚に減らす、または放射線（irradiated）タブレット等に変更してください。
*   **クリア速度の維持**: 霧のゲージを維持するため、ある程度サクサクと殲滅しながら素早く移動する必要があります。
*   **Chilled Ground (冷気地面) 対策**: 移動速度やジャンプが大幅に制限されるため非常にストレスになります。対策として **Legacy of Wonderlust**（冷気地面無効のユニーク靴）の導入を強く推奨します。

## 7. Spirit Walker (Twister Huntress) ビルドへの言及

Nick 2 がこのファームで使用している Spirit Walker ビルドに関する所感です。

*   **強み**: ダメージ、追加プロジェクトタイル、プロジェクタイル速度の恩恵が大きく、ファーム能力自体は非常に高いです。
*   **弱み**: **スペースキー（Dodge）の連打が必須**となる立ち回りが非常にストレスになります。
    *   **対策**: Nick 2 はこのストレスを回避するため、近いうちにアセンダンシーを Barrage や Frenzy チャージを主軸とした別のものに変更する予定とのことです。

## 8. Nick 2 の実証リターン例 (1マップあたりのドロップ)

動画内で紹介された、ある「下振れ気味」のマップでのドロップ内容：
*   **Simulacrum (完成品)**: 2個 (約 4 Divine)
*   **Orb of Annulment**: 2個 (200 Exalt ＝ 約 1.5 Divine)
*   **Chaos Orb**: 6個 (約 0.6 Divine)
*   **Master Domain Tablet / Abyss Tablet 等**: 約 2 Divine 相当
*   **各種カレンシー・アイテム**: 多数
*   **結果**: **合計 7〜8 Divine** のリターン（投資 1.5 Divine に対して大幅な黒字）。

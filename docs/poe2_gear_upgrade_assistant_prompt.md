# 🧠 PoE2 装備更新計画＆トレードURL自動生成プロンプト (AI指示書)

このファイルは、AI（Gemini等）に対してプレイヤーの現在の装備情報（`current_gear_summary.md` 等）を読み込ませ、ボトルネックの分析、装備の更新優先順位、および更新用アイテムを探すための `generate_trade_url.py` コマンド（またはトレードURL）を自動作成させるためのプロンプト定義です。

AIとチャットする際、以下の **「AIへの指示プロンプト」** をコピーし、**取得した装備サマリーの内容**を貼り付けて送信してください。

### 🤖 AIに実行してほしいタスク

1.  **生存能力（防御面）の現状分析**:
    *   装備から得られているライフと、三属性耐性（火・冷気・雷）の合計値を集計してください。
    *   現在のレベルに対して、ライフや耐性が十分か（特に各耐性75%を目指す上で何が何%不足しているか）を判定してください。
2.  **装備部位のボトルネック（弱点）特定**:
    *   装備スロット（頭・胴・手・足・指輪・アミュレット・ベルト・武器）を比較し、最もModの数値が低い、または重要なMod（ライフや耐性など）が欠落している「アップグレード優先度の高い部位」を3つ特定してください。
    *   それぞれの部位について、なぜ更新が必要なのか理由を説明してください。
3.  **アップグレード検索コマンドの生成**:
    *   特定したボトルネック部位を更新するために、`apps/97_poe/generate_trade_url.py` を呼び出すためのコマンドを部位ごとに組み立ててください。
    *   **コマンド組み立てのルール**:
        *   不足している耐性を補うために `--res-fire`, `--res-cold`, `--res-lightning` などの個別耐性オプション、または `--res` (総耐性) を適切に割り振ってください。
        *   防具には極力 `--life` オプション（レベルに応じた数値、例: Lv50代なら 40〜60以上）を付与してください。
        *   武器やアクセサリー（指輪・アミュレットなど）でビルドの方向性に合致するフラットダメージやキャスト速度・攻撃速度が必要な場合は、加点フラグ（`--phys`, `--cold`, `--fire`, `--lightning`, `--cast`, `--fire-spell`, `--cold-spell`, `--lightning-spell`）を追加してください。
        *   予算制限がある場合は `--max-price <Chaos額>` を付与してください。
        *   インスタントバイアウト限定の希望がある場合は `--instant-only` を付与してください。
